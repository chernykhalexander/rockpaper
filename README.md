# Rock Scissors Paper game

## Prerequisites

1. JDK 1.8
2. Maven
3. Access to the internet(to download dependencies)

## How to make a jar

1. run 
```
mvn clean package
```

## Run

1. cd cli/target

2. java -jar cli-1.0-SNAPSHOT-jar-with-dependencies.jar

## Play

In order to play the game type ```start```.

Now you can enter your turns, for example:

```r``` for ROCK

```sc``` for scissors

```p``` for paper

To stop the game enter ```stop```.

To get statistics enter ```gs```.

To quit type ```exit```.
